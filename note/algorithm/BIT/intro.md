# 樹狀數組
全名 Binary Index Tree  
看下面這個表就知道了

| 資料結構  | 查詢區間和         | 修改單點  |
| -------- | ----------------- | -------- |
| 陣列      |   O(n)           | O(1)     |
| 前綴和    |   O(1)           | O(n)     |
| BIT      |   O(log n)        | O(log n) |

查找跟修改的複雜度 BIT明顯優於於普通陣列及前綴和 
想快速查找區間和 又想快速修改的話 BIT是個不錯的選擇
