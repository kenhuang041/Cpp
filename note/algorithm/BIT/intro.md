# 樹狀數組
全名 Binary Index Tree  
複雜度優於於普通陣列及前綴和的資料結構

| 限制條件                  | 說明                                  |
| --------------------- | ----------------------------------- |
| `where T : class`     | T 必須是**參考型別**（例如 class、string、List） |
| `where T : struct`    | T 必須是**值型別**（例如 int、DateTime）       |
| `where T : new()`     | T 必須有**無參數建構子**                     |
